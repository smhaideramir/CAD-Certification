# CAD Basic Certification

This repository documents my CAD Basic Certificate project completed through IPEK at the Karlsruhe Institute of Technology (KIT).

The project involved working on a technical CAD task related to a gearbox assembly and producing a detailed technical drawing of an input shaft. I am including this repository as part of my technical portfolio to show basic CAD ability, technical drawing skills, and experience with engineering documentation.

## Skills demonstrated

- Basic CAD modelling workflow
- Reading and interpreting engineering drawings
- Producing a dimensioned technical drawing
- Understanding part geometry within a mechanical assembly
- Working with shafts, shoulders, diameters, grooves, and tolerances
- Basic engineering documentation

## Tools used

The CAD training was completed using Creo Parametric as part of the IPEK CAD Basic Certificate course.

## Project summary

The assignment focused on a gearbox-related component. My final submitted work was a technical drawing of an input shaft, including its geometry, dimensions, section view, and relevant manufacturing details.

Although this project is not directly a physics research project, it is relevant to experimental physics and engineering environments where technical drawings, mechanical components, instrument design, and clear documentation are important.

## Reflection

This certification helped me understand how mechanical parts are represented, dimensioned, and documented. It also gave me a basic foundation in CAD that may be useful in laboratory or experimental research settings, especially when working with measurement setups, mounts, holders, or simple mechanical components.
